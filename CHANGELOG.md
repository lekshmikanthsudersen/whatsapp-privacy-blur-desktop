# Changelog

## 1.0.1 - 2026-07-22

### Fixed

- Restores the unread-list position automatically after replying, with a single Resume unread list fallback action.
- Makes diagnostics recover visibly with Refresh instead of showing a blank window.
- Keeps the saved chat zoom in sync when Actual Size, Zoom In, or Zoom Out is used.
- Adds clear full-screen exit controls: F11, Escape, and a top-edge close button.
- Makes tray show/restore handling safer and adds an optional idle power saver for hidden or minimized windows.
- Adds a stable portable extraction folder and a ZIP distribution option for reliable taskbar pinning.
